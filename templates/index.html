👉🏽 Bấm tải phầm mềm (cấp quyền tải file nếu trình duyệt yêu cầu).
👉🏽 Mở file autolastwar.exe vừa tải (cấp quyền: "More info" -> "Run anyway" nếu windows yêu cầu).
👉🏽 Phần mềm yêu cầu nhập key, copy ID máy trên hộp thoại, gửi cho admin để nhận key.
👉🏽 Nhập key và kích hoạt để sử dụng.
👉🏽 Lưu ý: auto chỉ sử dụng cho game LastWar phiên bản PC tại lastwar.com
👉🏽 Sau khi kích hoạt thành công, một file license.key được tạo ra tự động. Để file license.key cùng thư mục với file autolastwar.exe (không cần nhập key những lần sau). Bắt đầy cày RSS thôi...
